# RL_PCB Workspace

请在本目录中 clone 外部仓库：

```powershell
git clone https://github.com/lukevassallo/rl_pcb.git
```

clone 后目录应类似：

```text
rl_pcb_workspace/
└── rl_pcb/
    ├── README.md
    ├── requirements.txt
    ├── setup.sh
    ├── install_tools_and_virtual_environment.sh
    ├── experiments/
    ├── tests/
    └── src/
```

注意：

- 该仓库官方安装说明偏向 Linux/Ubuntu。
- Windows 用户推荐使用 WSL2 + Ubuntu。
- 优先尝试 CPU-only 安装和测试。
- 所有运行过程、错误和解决办法都记录到 `reports/rl_pcb_report_template.md`。
