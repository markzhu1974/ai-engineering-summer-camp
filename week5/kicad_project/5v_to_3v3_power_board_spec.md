# 5V 转 3.3V 电源转换板规格说明

本规格用于 Week5 KiCad 实践。目标是画出一个简单、可检查、可导出制造文件的电源转换板。

## 1. 功能目标

输入：

```text
5V DC
```

输出：

```text
3.3V DC
```

## 2. 推荐电路

推荐使用 LDO 线性稳压器方案。

可选芯片：

- AMS1117-3.3
- AP1117-3.3
- LM1117-3.3

如果 KiCad 默认库中找不到完全一致型号，可以使用相近的三端 LDO 符号和封装完成练习。

## 3. 推荐元器件

| 编号 | 器件 | 建议值 | 说明 |
| --- | --- | --- | --- |
| U1 | LDO 稳压器 | AMS1117-3.3 | 5V 转 3.3V |
| C1 | 输入电容 | 10uF | 靠近 VIN |
| C2 | 输出电容 | 10uF | 靠近 VOUT |
| C3 | 去耦电容 | 0.1uF | 可选 |
| J1 | 输入端子 | 2Pin | 5V 和 GND |
| J2 | 输出端子 | 2Pin | 3.3V 和 GND |
| LED1 | 电源指示灯 | LED | 可选 |
| R1 | LED 限流电阻 | 1k | 可选 |

## 4. 网络命名建议

请使用清晰的网络名：

```text
+5V
+3V3
GND
```

## 5. PCB 设计要求

- 板子大小建议控制在 30mm x 30mm 左右。
- 输入电容靠近 LDO 输入脚。
- 输出电容靠近 LDO 输出脚。
- 电源线宽建议大于普通信号线。
- GND 可以使用铺铜。
- 必须运行 DRC，并解决明显错误。

## 6. 最终输出

你需要输出：

- 原理图截图。
- PCB Layout 截图。
- 3D Viewer 截图，可选。
- DRC 通过截图。
- Gerber 文件。
- Drill 文件。
