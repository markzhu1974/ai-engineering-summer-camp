# Week 3: LLM Chat Assistant

这是暑期 AI 工程实践训练营第三周的完整 Jupyter Notebook 工程。

本项目目标：开发第一个 LLM 应用，理解 Prompt Engineering、Few-shot Prompt、Prompt Template、LLM API 调用、聊天历史和简单命令行聊天助手的基本结构。

## 项目结构

```text
week3/
├── README.md
├── 运行指南.md
├── requirements.txt
├── .env.example
├── .vscode/
│   ├── extensions.json
│   └── settings.json
├── notebooks/
│   └── week03_llm_chat_assistant.ipynb
├── prompts/
├── results/
└── src/
```

## 主要文件

- `notebooks/week03_llm_chat_assistant.ipynb`：完整 LLM 应用实验 Notebook。
- `运行指南.md`：从安装 Python、VS Code、Jupyter 插件到运行 Notebook 的逐步指南。
- `.env.example`：API Key 配置示例。
- `requirements.txt`：项目依赖。

## 运行入口

请先阅读：

```text
运行指南.md
```

然后在 VS Code 中打开并运行：

```text
notebooks/week03_llm_chat_assistant.ipynb
```
