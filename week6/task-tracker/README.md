# Task Tracker 练习项目

这个目录用于 Week 6 的 Opencode + DeepSeek Python 动手实验。

开始时这里可以只有这个说明文件。请按照 `../docs/experiment_plan.md` 从实验 0 开始，让 Opencode 逐步生成 Python 命令行项目。

在实验 2 完成后，这个目录应该出现：

- `pyproject.toml` 或 `requirements.txt`
- `src/task_tracker/task.py`
- `src/task_tracker/service.py`
- `src/task_tracker/cli.py`
- `src/task_tracker/__main__.py`
- `tests/test_task_service.py`

常用验证命令：

```powershell
.\.venv\Scripts\Activate.ps1
python -m pytest
python -m task_tracker
```
