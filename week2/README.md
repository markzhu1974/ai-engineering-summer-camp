# Week 2: MNIST Image Classifier

这是暑期 AI 工程实践训练营第二周的完整 Jupyter Notebook 工程。

本项目目标：使用 PyTorch 完成第一个图像分类项目，理解 Tensor、Dataset、DataLoader、MLP、Loss、Optimizer、训练循环、评估和模型保存。

## 项目结构

```text
week2/
├── README.md
├── 运行指南.md
├── requirements.txt
├── .vscode/
│   ├── extensions.json
│   └── settings.json
├── data/
├── figures/
├── models/
├── notebooks/
│   └── week02_mnist_mlp_classifier.ipynb
└── results/
```

## 主要文件

- `notebooks/week02_mnist_mlp_classifier.ipynb`：完整 PyTorch 图像分类实验 Notebook。
- `运行指南.md`：从安装 Python、VS Code、Jupyter 插件到运行 Notebook 的逐步指南。
- `requirements.txt`：项目依赖。

## 运行入口

请先阅读：

```text
运行指南.md
```

然后在 VS Code 中打开并运行：

```text
notebooks/week02_mnist_mlp_classifier.ipynb
```
