# Week 1: House Price Prediction

这是暑期 AI 工程实践训练营第一周的完整 Jupyter Notebook 工程。

本项目目标：通过一个房价预测任务，完整走通 AI 工程中的数据读取、EDA、特征工程、模型训练、模型评估、可视化和模型保存流程。

## 项目结构

```text
week1/
├── README.md
├── 运行指南.md
├── requirements.txt
├── .vscode/
│   ├── extensions.json
│   └── settings.json
├── data/
├── figures/
├── models/
├── notebooks/
│   └── week01_house_price_prediction.ipynb
└── results/
```

## 主要文件

- `notebooks/week01_house_price_prediction.ipynb`：完整实验 Notebook。
- `运行指南.md`：从安装 Python、VS Code、Jupyter 插件到运行 Notebook 的逐步指南。
- `requirements.txt`：项目依赖。

## 运行入口

请先阅读：

```text
运行指南.md
```

然后在 VS Code 中打开并运行：

```text
notebooks/week01_house_price_prediction.ipynb
```
