注意，直接下载的原始代码因为其用的python版本比较旧，在直接运行的时候会报错。我们需要修改以下地方：

### （1）第一处要修改的地方

```Python
% matplotlib inline

# The above command sets the backend of matplotlib to the 'inline' backend. 
# It means the output of plotting commands is displayed inline.
```
我们需要将%后面的空格删除，变成

```Python
%matplotlib inline
```

### （2）第二处要修改的地方
```Markdown
## About the dataset

The data set has been imported from the econometrics website with the following url:-

http://www.econometrics.com/intro/sales.htm

This data set contains Sales and Advertising expenditures for a dietary weight control product. It contains monthly data for 36 months. The variables in this data set are Sales and Advertising.
```

```Python
# Import the data

url = "C:/project_datasets/SALES.txt"
df = pd.read_csv(url, sep='\t', header=None)
```

在这里，原始代码中从上面的链接http://www.econometrics.com/intro/sales.htm 下载了数据文件SALES.txt, 并且保存在了C:/project_datasets/SALES.txt。

我们需要到这个网站手动下载SALES.txt。在点开网页后看到这样一行

A listing of the data set ( filename: SALES.txt ) follows.

在SALES.txt的链接上点右键，选择“将链接另存为”，把它保存到本地。

把SALES.txt文件移动到handson-ml3目录下，和SLRProject.ipynb 文件在一起。

```Text
handson-ml3/
    |-- SLRProject.ipynb
    |-- SALES.txt
```

然后把上面代码中的url更新一下，同时读取csv的函数的参数也需要更新，变成下面这样：

```Python
url = "SALES.txt"
df = pd.read_csv(url, sep=r"\s+", header=None, on_bad_lines="skip")
```

因为数据文件和代码文件在同一级目录下，所以相对路径就是当前目录了。

### 2. 修改数据处理

* 缺失值的处理

首先修改数据文件SALES.txt。当前每一行两列数据都是有内容的。我们任意删掉某几个数值，像下面这样：

```Text
    12.0   15.0
    20.5   16.0
    21.0   
    15.5   27.0
    15.3   
    23.5   49.0
    24.5   21.0
    21.3   22.0
```
然后重新Run All,

就会看到print(df.head())输出的数据中就有空值（NaN）
```Text
      0     1
0  12.0  15.0
1  20.5  16.0
2  21.0  18.0
3  15.5   NaN
4  15.3  21.0
```

所以我们需要进行空值处理。实际上会有多种空值处理的方法，例如去掉有空值的这一行数据，用平均值填补空值，用上一个值填补空值等等。

这里我们用平均值填补。

在pandas columns attribute文字的前面，加入一个代码单元，输入下面的代码：

```Python
df = df.fillna(df.mean())
```

然后再Run all。这时看到之前有NaN空值的地方就被数值填充了。



---

### 3. 改模型参数

模型参数的修改是进行模型调参的必须手段。

这里我们可以尝试做下面的调整，然后重新Run All。

1）修改 Train/Test Split 比例：

找到代码：

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.33, random_state=42)

修改动作：

将 test_size=0.33 改为 test_size=0.1（10%数据测试）或 test_size=0.5（50%数据测试）。

观察：

重新运行后续的 fit 和评分单元格，记录并对比不同切分比例下R平方的变化。

2） 改变随机种子 random_state：

修改动作：

将 random_state=42 修改为其他任意整数（如 100），或者直接将这个参数删掉。

观察：

连续点击运行该单元格多次，看看每次得到的模型斜率、截距以及 R平方是否还完全一致？体会随机数种子的作用。


### 4. 扩展思考：能把线性回归模型换一个更好的算法模型吗？

线性回归模型是一个最基本的机器学习模型，我们也听过很多其它的机器学习模型，例如决策树，随机森林，GBDT等等。在实际的项目中，可以用它们替换线性回归模型，通过实际数据训练来挑出性能最好的算法模型。只需要在sklearn的机器学习库中选择相应的模型库就可以使用了，例如我们用的线性回归使用的是LinearRegression 模型。这个本次训练营就不去尝试了，在学习了机器学习课程之后就可以应用。

## 必做任务

### 1. 切分数据

新加一个代码单元，写入下面这一行代码执行

```python
import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.model_selection import train_test_split

df = pd.read_csv("../data/housing.csv")

X = df.drop("target", axis=1)
y = df["target"]
X = X.fillna(X.mean())

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y, test_size=0.2, random_state=42
)
```
